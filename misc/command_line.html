Command line options¶

MotionON can be started from the command line.

Usage¶

> motionon.exe
    [-name <string>] [-hideExplorer] [-workspace <path>]
    [-video <path>] [-speed <0..200>] [-stretch]

Options¶

-name <string>

Name of this instance of MotionON. Used in the window title and to select a preference file.

-hideExplorer

The explorer panel will not be visible. Default: false.

-workspace <path>

Path to a MotionON workspace XML file. This overrides other video options. To create a workspace file use the menu Option ‣ Workspace ‣ Export workspace.

-video <path>

Path to a video to load.

-speed <0..200>

Playback speed to play the video, as a percentage of its original framerate. Default: 100.

-stretch

The video will be expanded to fit the viewport. Default: false.

Examples¶

> motionon.exe -video test.mp4 -stretch
> motionon.exe -video test.mp4 -speed 50
> motionon.exe -name Replay -workspace myReplayWorkspace.xml
